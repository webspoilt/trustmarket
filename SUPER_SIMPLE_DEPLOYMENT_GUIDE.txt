STEP 1: CREATE GITHUB ACCOUNT (5 minutes)
===========================================

1. Open your browser (Chrome, Firefox, Safari)
2. Type: github.com
3. Click "Sign up"
4. Choose username (like: yourname2024)
5. Enter email and password
6. Click "Create account"
7. Check your email and click the link

‚úÖ DONE! You now have a GitHub account

===========================================
STEP 2: UPLOAD YOUR CODE (10 minutes)
===========================================

1. Go to github.com (login if needed)
2. Click the green "New" button
3. Fill in:
   - Repository name: trustmarket
   - Description: My awesome marketplace app
   - Make it Public (free)
   - Don't check "Initialize with README"
4. Click "Create repository"

5. You'll see a page with code commands - IGNORE THEM!
6. Scroll down and click "uploading an existing file"
7. Drag and drop ALL your TrustMarket files
8. Click "Commit changes"

‚úÖ DONE! Your code is now in GitHub

===========================================
STEP 3: FRONTEND DEPLOYMENT - VERCEL (5 minutes)
===========================================

1. Open new tab: vercel.com
2. Click "Sign up with GitHub"
3. Give permission (click "Authorize")
4. Click "Add New" ‚Üí "Project"
5. Find your "trustmarket" repo and click "Import"
6. Vercel will detect it's a React app automatically
7. Click "Deploy" (green button)

üéâ FRONTEND IS LIVE! 
You now have a URL like: https://trustmarket-abc123.vercel.app

===========================================
STEP 4: BACKEND DEPLOYMENT - RENDER (5 minutes)
===========================================

1. Open new tab: render.com  
2. Click "Sign up with GitHub"
3. Give permission
4. Click "New+" ‚Üí "Web Service"
5. Find your "trustmarket" repo and click "Connect"
6. Fill in settings:
   - Name: trustmarket-api
   - Runtime: Node
   - Build Command: npm install
   - Start Command: npm start
7. Click "Create Web Service"

üéâ BACKEND IS LIVE!
You now have a URL like: https://trustmarket-api.onrender.com

===========================================
STEP 5: DATABASE SETUP - MONGODB ATLAS (5 minutes)
===========================================

1. Open new tab: mongodb.com/atlas
2. Click "Start Free"
3. Fill out signup form
4. Choose "Free" plan (already selected)
5. Click "Create Cluster"
6. Wait 2-3 minutes for cluster to be ready
7. Click "Connect" on your cluster
8. Click "Add IP Address"
9. Click "Allow access from anywhere" (for now)
10. Click "Confirm"
11. Click "Create Database User"
12. Enter username: trustmarket_user
13. Enter a strong password (write it down!)
14. Click "Create User"
15. Click "Connect your application"
16. Copy the connection string (starts with mongodb+srv://)

‚úÖ DATABASE IS READY!

===========================================
STEP 6: PICTURE STORAGE - CLOUDINARY (3 minutes)
===========================================

1. Open new tab: cloudinary.com
2. Click "Sign up for free"
3. Fill out form (use same email as GitHub)
4. Choose "Free" plan
5. Once in dashboard, look for:
   - Cloud Name (like: my-cloud-name)
   - API Key
   - API Secret
6. Write these down (you'll need them!)

‚úÖ CLOUDINARY IS READY!

===========================================
STEP 7: CONNECT EVERYTHING (10 minutes)
===========================================

BACKEND SETUP:
1. Go to Render.com ‚Üí Your service
2. Click "Environment" tab
3. Add these one by one:

   PORT = 5000
   MONGODB_URI = (paste your MongoDB connection string here)
   JWT_SECRET = my-super-secret-key-12345
   REFRESH_TOKEN_SECRET = my-refresh-secret-key-67890
   CLOUDINARY_CLOUD_NAME = (your cloud name)
   CLOUDINARY_API_KEY = (your API key)
   CLOUDINARY_API_SECRET = (your API secret)
   CLIENT_URL = (your Vercel URL from Step 3)

4. Click "Save Changes"
5. Click "Manual Deploy" ‚Üí "Deploy latest version"

FRONTEND SETUP:
1. Go to Vercel.com ‚Üí Your project
2. Click "Settings" ‚Üí "Environment Variables"
3. Add these:

   VITE_API_URL = (your Render URL)
   VITE_CLOUDINARY_CLOUD_NAME = (your cloud name)

4. Click "Save"
